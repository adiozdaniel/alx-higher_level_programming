## Linter

### pycodestyle

Make sure you have pycodestyle installed!
To install use:

* ```$ pip install pycodestyle```
To check for syntax errors:

* ```pycodestyle --first *.py```
* ```pycodestyle --show-source --show-pep8 *.py```
* ```pycodestyle --statistics -qq *.py```
